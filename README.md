# LogistiChain Backend — Estructura tipo EMSX (sin paquete 'domain')

Paquete raíz: `com.app.logistichain`
Subpaquetes: `controllers`, `dtos`, `entities`, `exceptions`, `mapperdtos`, `repositories`, `services`, `servicesimpls`.

Cada entidad posee su DTO, Mapper, Service y ServiceImpl. Controladores CRUD listos.
Configuración en `application.properties`.
